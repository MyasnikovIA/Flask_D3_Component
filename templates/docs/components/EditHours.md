# EditHours

Компонент EditHours представляет собой поле для ввода интервала времени в часах.

В БД значение хранится в виде числа с плавающей точкой (количество часов), например, интервал 1ч 30 мин = 1.5

Все основные свойства наследуются от компонента Edit.

## Примеры синтаксиса

```xml
 <cmpEditHours name="field_hours_Ctrl" width="100"/>
```

## Свойства компонента EditHours

|Название|Значение|Тип|Значение по умолчанию|
|---|---|---|---|
|format|Опции форматирования. Подробнее смотри метод onformat компонента Edit|string|{toType : 'hours'}|
|mask_type|Тип маски компонента Mask|string|hoursminutes|
